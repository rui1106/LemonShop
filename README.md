# LemonShop


## 介绍

django分离完成后端的一个web项目



## 项目说明
这个项目是以django，jwt,mysql,redis等技术实现的一个web项目



## 项目内容




## 实现的功能
1. 登录，注册
2. 商品显示,购物车的显示,添加和删除
3. 用户中心添加收货地址,显示,删除,修改
4. 结算页面显示购物车商品和用户添加的收货地址
5. 支付功能,用到了支付宝
